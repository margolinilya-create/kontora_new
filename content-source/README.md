# Content source of truth

Этот каталог — единственное место, где живёт «сырой» текст сайта.

## Файлы

- `pdf-extract.txt` — точная транскрипция PDF «Контора нью», 13 страниц. Любые изменения текста на сайте начинаются **здесь**.

## Правила

1. **Не править `lib/content/**` напрямую.** Сначала правим `pdf-extract.txt`, потом переносим правку в типизированный контент в `lib/content/**`.
2. **CI гейт.** `scripts/check-copy.ts` сравнивает строки из `lib/content/**` с `pdf-extract.txt`. Расхождение → PR красный.
3. **Как обновить после правок владельца:**
   ```bash
   # 1. обновите pdf-extract.txt под свежую версию PDF
   # 2. обновите соответствующий файл в lib/content
   # 3. npm run check:copy  → должно быть зелёно
   # 4. npm run build       → должно собраться
   # 5. commit
   ```

## Почему не CMS?

Осознанное решение v3-плана. CMS добавляет время, риск, стоимость. Владелец редактирует контент через GitHub web editor — этого достаточно до появления реальной потребности.
